# Kanban CRM Apex - Tema WordPress

## Instalação

1. Baixe o arquivo: kanban-crm-apex-theme.tar.gz

2. No seu WordPress:
   - Acesse: Aparência > Temas
   - Clique em "Adicionar Novo"
   - Clique em "Enviar Tema"
   - Escolha o arquivo kanban-crm-apex-theme.tar.gz
   - Clique em "Instalar Agora"
   - Ative o tema

## Como Usar

1. Após ativar o tema, acesse sua página inicial
2. Clique em "Upload CSV" para carregar seus dados do CRM
3. Use o botão "Dashboard" para ver análises gráficas
4. Os dados ficam salvos no navegador até você carregar um novo arquivo

## Formato do CSV

O arquivo CSV deve conter as colunas:
- id
- name
- phone  
- email
- type
- value
- pipeline_stage (new, prospeccao, follow_up_a, follow_up_b, follow_up_c, agenda_reuniao, follow_up_d, follow_up_e, won, lost)

## Funcionalidades

✓ Visualização Kanban horizontal com scroll lateral
